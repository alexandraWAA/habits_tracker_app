# Habits Tracker - Трекер полезных привычек

Бэкенд-часть SPA веб-приложения для трекинга полезных привычек, вдохновленная книгой Джеймса Клира «Атомные привычки».

## 🚀 Технологии

- Python 3.11+
- Django 4.2.7
- Django REST Framework 3.14.0
- PostgreSQL 15
- Redis / Celery
- Docker / Docker Compose
- GitHub Actions (CI/CD)
- Telegram Bot API
